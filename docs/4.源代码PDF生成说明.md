# 源代码PDF整理指南

软著登记要求提交**源代码PDF**，通常要求：
- **≥10页**（建议60页以上更稳妥）
- **最后400行以上**（要求最后连续400行不能是空行或纯注释）
- **前30页 + 后30页** 或 **前60页**
- 代码中的**软件名称、版本号**必须与说明书一致

## 方法二：手动步骤

### Step 1：创建输出目录

```bash
mkdir -p /Users/fan/www/AI/erp-saas/docs/source_code
```

### Step 2：提取全部源代码（不含node_modules）

```bash
cd /Users/fan/www/AI/erp-saas

# 提取前端代码并排序（让关键文件排在前面）
find apps/web/src -type f \( -name "*.ts" -o -name "*.tsx" \) \
  | sort \
  > docs/source_code/web_files.txt

# 提取后端代码并排序
find apps/api/src -type f -name "*.ts" \
  | sort \
  > docs/source_code/api_files.txt
```

### Step 3：用Python生成PDF兼容的代码文档

```python
# 保存为 docs/source_code/make_pdf.py 运行
import os

output_file = "/Users/fan/www/AI/erp-saas/docs/source_code/云ERP零售批发版_源代码.pdf.txt"

# 合并所有文件
with open(output_file, 'w', encoding='utf-8') as out:
    out.write("=" * 70 + "\n")
    out.write("  云ERP（零售批发版） v1.0.0\n")
    out.write("  著作权人：________________\n")
    out.write("  源程序量：约 8700 行\n")
    out.write("=" * 70 + "\n\n")

    # 前端代码
    out.write("=" * 70 + "\n")
    out.write("  前端源代码 (apps/web/src)\n")
    out.write("=" * 70 + "\n\n")

    with open("apps/web_files.txt") as f:
        for line in f:
            line = line.strip()
            if not line:
                continue
            rel_path = line.replace("apps/web/src/", "")
            out.write(f"\n// ========== {rel_path} ==========\n")
            try:
                with open(line, 'r', encoding='utf-8') as code:
                    content = code.read()
                    out.write(content)
                    out.write("\n")
            except Exception as e:
                out.write(f"// [文件读取失败: {e}]\n")

    # 后端代码
    out.write("\n" + "=" * 70 + "\n")
    out.write("  后端源代码 (apps/api/src)\n")
    out.write("=" * 70 + "\n\n")

    with open("apps/api_files.txt") as f:
        for line in f:
            line = line.strip()
            if not line:
                continue
            rel_path = line.replace("apps/api/src/", "")
            out.write(f"\n// ========== {rel_path} ==========\n")
            try:
                with open(line, 'r', encoding='utf-8') as code:
                    content = code.read()
                    out.write(content)
                    out.write("\n")
            except Exception as e:
                out.write(f"// [文件读取失败: {e}]\n")

print(f"代码文档已生成：{output_file}")
```

### Step 4：转换为PDF

1. 将生成的 `.txt` 文件内容全选复制
2. 打开 Word，粘贴，保持等宽字体（如 Courier New，字号10-11pt）
3. 调整页边距（上2cm，下2cm，左2.5cm，右2cm）
4. 导出为PDF（文件 → 导出 → PDF）
5. 确保PDF页数 ≥ 60页

## 注意事项

1. **最后400行**必须是有效代码，不能是空行或纯注释行
2. **软件名称**（云ERP）和**版本号**（v1.0.0）必须在代码注释中体现
3. 建议在每个源文件头部加上：
   ```typescript
   /**
    * 云ERP（零售批发版）v1.0.0
    * 著作权人：________________
    */
   ```
4. PDF页数建议 ≥ 60页，不要只有10-15页，太少容易被质疑
5. 源代码中不要包含 `node_modules`、`dist`、`build` 等目录
